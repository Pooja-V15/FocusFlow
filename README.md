# 🚀 FocusFlow

## AI-Powered Productivity & Study Planner

> **Focus Better. Study Smarter.**

**Developed by Pooja V**

FocusFlow is an AI-powered productivity platform that combines task management, smart planning, Pomodoro focus sessions, productivity analytics, and AI-powered recommendations into one modern workspace. It helps users stay organized, build consistent habits, and improve daily productivity through an intuitive and visually engaging interface.

---

# 📖 Overview

FocusFlow is designed to simplify productivity by bringing planning, task management, focus sessions, and progress tracking into a single application. Instead of relying on multiple tools, users can organize their workflow, monitor productivity, and stay motivated from one centralized dashboard.

---

# 💡 Problem Statement

Managing tasks, schedules, and focus across multiple applications often leads to distractions and reduced productivity. FocusFlow solves this by providing an all-in-one workspace with intelligent planning, analytics, and gamified motivation.

# ✨ Features

### 🎯 Smart Dashboard
View study hours, focus score, pending tasks, completed goals, and productivity insights from a single dashboard.

### 🍅 Focus Mode
Boost concentration with customizable Pomodoro timers, ambient sounds, and focus statistics.

### 📅 Planner
Organize daily, weekly, and monthly schedules with reminders, priorities, and time-blocking.

### ✅ Task Management
Create, organize, prioritize, and track tasks with categories, labels, and progress indicators.

### 📊 Analytics
Monitor productivity through interactive charts, weekly reports, and goal tracking.

### 🏆 Gamification
Stay motivated with XP, streaks, badges, levels, and achievement rewards.

### 🤖 AI Productivity Assistant
Receive AI-powered study plans, task prioritization, and personalized productivity suggestions.

### 👤 User Profile
Manage your profile, achievements, preferences, and productivity statistics.

# 🚀 Getting Started

## Clone the Repository

```bash
git clone https://github.com/Pooja-V15/FocusFlow.git
cd FocusFlow
```

## Install Dependencies

```bash
npm install
```

## Run the Development Server

```bash
npm run dev
```
